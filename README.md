# Java University Tasks

This repository contains Java programming assignments completed as part of my university coursework.

## Structure

Each assignment is organized in a separate folder:

```text
task-01/
task-02/
task-03/
...
```

Each task may contain multiple Java classes and supporting files depending on the assignment requirements.

## Technologies

* Java
* IntelliJ IDEA

## Purpose

The purpose of this repository is to store, organize, and track the development of university Java assignments using Git and GitHub.
